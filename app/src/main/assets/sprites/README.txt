Drop OpenGameArt sprite sheets in this folder.

Suggested files to start with:
- kerry_sheet.png
- tree_sheet.png
- effects_sheet.png
- background_sheet.png

Recommended default frame sizes (can be adjusted in SpriteSheetSupport.kt):
- Kerry: 32x32
- Tree: 64x64
- Effects: 16x16
- Background tiles: 16x16
